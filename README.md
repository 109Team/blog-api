### 博客WEB API
___
#### 技术栈
> 使用`express` + `mongoose`构建，
> swagger使用`swagger-ui-express`

#### 实现功能
    1. 文章增删改查
    2. 文章标签
    3. 文章分类
#### 启动方式
1. 
``` javascript
git clone
```
2.
``` javascript
cd blog-api
npm install
```
3.
``` javascript
MacOS or Linux
DEBUG=blog-api:* npm start

windows
set DEBUG=blog-api:* npm start
```

4.备注
需本地安装mongodb服务
